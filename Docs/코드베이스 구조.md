## 게임오버 시스템 플로우

```
[Server]
  GoInGameServerSystem
    └─ Hero 생성 + UserAliveState (Connection 엔티티에 부착)

[Server] SimulationSystemGroup
  HeroDeathDetectionSystem (UpdateBefore: ServerDeathSystem)
    ├─ 감지: Health <= 0 + HeroTag
    ├─ UserAliveState.IsAlive = false
    ├─ 전송: HeroDeathRpc (해당 유저)
    └─ 전송: GameOverRpc (모든 유저 사망 시, 1회 한정)

[Client] SimulationSystemGroup
  GameOverReceiveSystem
    ├─ HeroDeathRpc 수신 → UserState.CurrentState = Dead
    └─ GameOverRpc 수신 → GameOverEvents.RaiseGameOver()

[Client] MonoBehaviour
  GameOverEvents (정적 이벤트)
    └─ OnGameOver → GameOverPanelController.ShowGameOverPanel()
```
