# Project-SOS

## Required Pakages
### How to Install?
1. Open the Package Manager window (Window > Package Manager)
2. Add the package by its name
### Required Pakage Names
1. Entities: com.unity.entities
2. Entities Graphics: com.unity.entities.graphics
3. Netcode For Entities: com.unity.netcode
4. Multiplayer Play Mode: com.unity.multiplayer.playmode

## Assets > Project Settings 
 1. Editor > Enter Play Mode Setting > When entering Play Mode > Do not reload Domain or Scene
 2. Player > Run in Background Check
